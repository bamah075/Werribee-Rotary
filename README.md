# Werribee Rotary Club

Welcome to the official repository for the Werribee Rotary Club website!

## About

This repository contains the website for Werribee Rotary Club, a community service organization dedicated to making a positive impact in the Werribee area.

## Features

- Responsive design for all devices
- Clean, professional layout
- Easy to navigate sections
- Community-focused content

## Getting Started

To view the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/bamah075/Werribee-Rotary.git
   ```

2. Open `index.html` in your web browser

## Publishing to GitHub Pages

To publish this website:

1. Go to **Settings** → **Pages**
2. Select **Deploy from a branch**
3. Choose **main** branch and **/ (root)** folder
4. Click **Save**

Your site will be available at: `https://bamah075.github.io/Werribee-Rotary/`
